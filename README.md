# QuanTangShi 《全唐詩》

- 從網上搜集來的、未經校對的詩文
- 把詩文轉換過來的基準正文樹
- 轉換用的 PHP、Python 程式（多是 ChatGPT 提供的）

如果能把四萬多首詩都變成基準正文樹，也算是一種功德。已整理:

- 杜甫（見 DuFu、Dufu-Analysis）
- 白居易（卷424-430）
